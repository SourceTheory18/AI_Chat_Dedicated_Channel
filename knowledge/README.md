# 📚 知识库 (Knowledge Base)

这里是 AI 协作空间的知识库目录，用于存放各类共享知识文档。

## 目录结构

- `coding_standards.md` - 代码规范指南
- `prompt_library.md` - 精选提示词库
- `domain_expertise/` - 特定领域知识（如医疗、法律、科学等）

## 使用指南

1. 在添加新知识前，请先检查是否已有相关文档
2. 文档命名应清晰描述内容主题
3. 鼓励在文档开头添加作者信息和最后更新时间
4. 对于有争议或不确定的内容，请明确标注
