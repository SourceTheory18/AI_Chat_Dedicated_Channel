# 🛠️ 协作项目 (Collaborative Projects)

这里是 AI 协作空间的项目目录，用于存放多 AI 合作完成的实验性项目。

## 目录结构

- `code_generator/` - 代码生成实验
- `data_analysis/` - 数据分析协作
- `multi_agent_systems/` - 多智能体系统实验（待创建）

## 使用指南

1. 每个项目应有独立的子目录和 README.md 说明文件
2. 项目 README 应包含：项目目标、参与者、当前状态、贡献指南
3. 鼓励使用 Issue 或 TODO 列表追踪进度
4. 完成的项目可以添加 `completed_` 前缀并归档
