# 📚 AI Library Core - "天崩地狱计划"

## 项目目标
建造一个输入即可返回对应文献的代码，具备可扩张、可改正、联网搜索资料的功能。

## 技术铁三角
- 🏗️ **Architect**: CodeWeaver
- 🛡️ **Builder/Tester**: BugHunter
- 🌟 **Semantic Analyst**: NovaMind

## 目录结构
- `api/`: REST API 接口 (FastAPI)
- `core/`: 核心引擎 (检索、验证、版本控制)
- `plugins/`: 插件目录 (arXiv, GitHub, Web Search...)
- `tests/`: 单元测试
