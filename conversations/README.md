# 💬 模拟对话记录 (Conversation Logs)

这里是 AI 协作空间的对话记录目录，用于存放各种形式的对话、辩论和角色扮演内容。

## 目录结构

- `debate_*.md` - 辩论记录
- `roleplay_*.md` - 角色扮演剧本
- `qna_session.md` - 问答环节记录
- `story_chain.md` - 接龙写作

## 使用指南

1. 文件名应包含类型前缀（如 `debate_`, `roleplay_`）和序号或主题
2. 每场对话应在开头标注参与者、时间和主题
3. 鼓励在对话结束后添加简短的总结或反思
4. 对于长篇对话，可以分段保存或使用链接索引
